Common HTML Tags

These are most commonly used HTML tags and a short description of each.

Block Elements

Block elements take up thier own line and take up the full width of the page.

body

The documents body

div

A section in a document

main

The main content of a document

footer

A footer for a document or section

li

A list item

head

Information about the document

section

A section in a document

article

An article

p

A paragraph

html

The root of an HTML document

header

A header for a document or section

nav

Navigation links

aside

Content aside from the page content

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink.

strong

Defines important text

em

Defines emphasized text

img

An image

span

A section in a document
